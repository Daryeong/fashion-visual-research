VELUNE

AI Fashion Brand
Web Project

AI를 활용해 가상의 패션 브랜드 VELUNE을 기획하고, 대량 엑셀 프롬프트 기반 이미지 생성, 상품 구조, 카테고리 설계, 웹 화면 구성까지 HTML/CSS/JS 기반 웹사이트로 완성한 프로젝트입니다.

2026.06.01 — 2026.06.11

Timeline / Process

Project Process

  1. 06.01

    Direction & Reference Research

    브랜드 이미지, 디자인 프롬프트, 오픈소스 디자인 자료를 조사하고 Dover Street Market, MSCHF, MACHINE-A 등의 사례를 참고해 VELUNE의 브랜드 방향과 웹 구조를 구체화했습니다.

  2. 06.02 — 06.04

    AI Image Generation

    대량 엑셀 프롬프트를 활용해 상품 이미지와 모델 이미지를 생성했습니다. 생성 결과를 비교하며 VELUNE의 브랜드 무드에 맞는 이미지 방향을 정리했습니다.

  3. 06.06 — 06.09

    Web Function & Structure Planning

    쇼핑몰 웹사이트에 필요한 기본 구조와 기능을 설계했습니다. 상품 목록, 상품 상세 페이지, 카테고리 메뉴, 검색, 필터, 장바구니, 찜 기능, Quick View 등 주요 쇼핑 흐름을 정리하고, 이를 HTML/CSS/JS 기반의 웹사이트 결과물로 완성했습니다.

  4. 06.09 — 06.11

    Portfolio Documentation & Shopping Mall Refinement

    완성된 쇼핑몰 웹사이트를 바탕으로 포트폴리오 구성을 정리하고, 상품명, 카테고리 체계, 상세 페이지, 이미지 구성, 메뉴 구조를 보완하며 최종 결과물의 완성도를 높였습니다.

Project Flow

컨셉초안에서 최종 결과까지, 각 단계의 이미지와 판단 기준을 크게 확인할 수 있도록 구성했습니다.

컨셉초안

컨셉 초안입니다. 미래적인 분위기, 패션 쇼핑몰의 판매 화면, 어두운 톤과 밝은 톤의 차이를 비교하며 방향을 잡았습니다.

Goal웹 전체 분위기를 먼저 잡기
Process레퍼런스 정리 → 톤 비교 → 화면형 이미지로 확장
Result패션 웹에 맞는 회색 중심의 차분한 방향 선택
ToolsImage Generation · Mood Research

Reference Analysis

RunwayML, x.ai, Linear의 디자인 구조를 확인한 뒤, 패션 쇼핑몰 화면에 적용할 수 있는 큰 이미지, 적은 텍스트, 정제된 카드 UI 방향을 선별했습니다.

초기에는 미래적인 화면에 가까웠지만, 실제 판매 화면과 브랜드 무드가 함께 보이도록 점차 방향을 수정했습니다.

초기방향

전체 웹 페이지 이미지 초안본입니다. 홈, 메뉴, 검색, 장바구니, 구매 화면 등 실제 쇼핑몰 흐름을 화면 단위로 나눠 확인했습니다.

Goal쇼핑몰 기본 흐름을 화면으로 빠르게 확인하기
Process홈 화면 → 메뉴 → 검색 → 장바구니 → 구매 화면 순서로 구성
Result사용 흐름은 잡혔지만, 톤과 정리 방식은 수정이 필요하다고 판단
ToolsHTML · CSS · Screen Capture

Draft Review

초안에서는 기능 흐름을 먼저 확인했습니다. 검색, 장바구니, 찜, 카테고리 이동은 작동했지만 화면이 복잡하게 보였고, 패션 브랜드의 첫인상보다 기능 요소가 먼저 보이는 문제가 있었습니다.

이후 최종 방향에서는 기능 수를 늘리는 것보다 홈 화면, 메뉴 구조, 상품 이미지 배치가 브랜드 무드와 연결되도록 정리했습니다.

최종방향

전체 웹 페이지 이미지 수정본입니다. 초안에서 보인 어색한 부분을 줄이고, 홈 화면과 메뉴 구조가 더 분명하게 보이도록 정리했습니다.

Goal브랜드 느낌과 사용 흐름이 같이 보이는 웹 화면 만들기
Process초안 비교 → 메뉴 구조 정리 → 홈 화면 완성도 조정 → 세부 화면 연결
Result홈, 카테고리, 검색, 위시리스트, 장바구니 흐름을 한 세트로 정리
ToolsHTML · CSS · JavaScript

Implementation Summary

Prompt Sheet

패션 액세서리와 소품 이미지를 여러 장 만들 수 있도록, 프롬프트가 자동으로 조합되는 엑셀 템플릿을 제작했습니다.

고급 편집숍 이미지처럼 정제된 구성은 살리되, 조금 더 개성 있고 격이 느껴지는 럭셔리 브랜드 무드를 참고했습니다. 제품 유형, 소재, 색감, 조명, 배경, 분위기를 조합해 다양한 이미지 프롬프트를 만들 수 있도록 구성했습니다.

컬러는 블랙, 차콜, 실버, 버건디, 포슬린 화이트, 산성 포인트 컬러를 중심으로 잡았습니다.

참고한 무드는 다음과 같습니다.

SSENSE, Mytheresa, NET-A-PORTER, LN-CC

브랜드 무드
Rick Owens, Maison Margiela, Yohji Yamamoto, The Row

고급스럽지만 뻔하지 않은 무드
