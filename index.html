Project Overview

회색 톤의 패션 웹을 기준으로, 컨셉 이미지부터 실제 웹 화면과 상품 이미지까지 순서대로 정리했습니다.

Goal / Process / Result / Tools

Goal패션 쇼핑몰에 맞는 분위기와 화면 구성을 찾고, 실제 홈 화면에 쓸 수 있는 이미지 기준을 정리하기
Process컨셉 초안 제작 → 웹 화면 초안 → 수정본 비교 → 상품/모델 이미지 성공·실패 분류 순서로 작업을 정리했습니다.
Result무드보드, 웹 화면, 상품 이미지, 모델 이미지가 한 흐름으로 보이도록 정리
ToolsHTML · CSS · JS · AI · Prompt Sheet

01. Concept Draft

컨셉 초안입니다. 미래적인 분위기, 패션 쇼핑몰의 판매 화면, 어두운 톤과 밝은 톤의 차이를 비교하며 방향을 잡았습니다.

Goal웹 전체 분위기를 먼저 잡기
Process레퍼런스 정리 → 톤 비교 → 화면형 이미지로 확장
Result패션 웹에 맞는 회색 중심의 차분한 방향 선택
ToolsImage Generation · Mood Research

02. Web Draft

전체 웹 페이지 이미지 초안본. 홈, 메뉴, 검색, 장바구니, 구매 화면 등 실제 쇼핑몰 흐름을 화면 단위로 나눠 확인했습니다.

Goal쇼핑몰 기본 흐름을 화면으로 빠르게 확인하기
Process홈 화면 → 메뉴 → 검색 → 장바구니 → 구매 화면 순서로 구성
Result사용 흐름은 잡혔지만, 톤과 정리 방식은 수정이 필요하다고 판단
ToolsHTML · CSS · Screen Capture

03. Web Final

전체 웹 페이지 이미지 수정본. 초안에서 보인 어색한 부분을 줄이고, 홈 화면과 메뉴 구조가 더 분명하게 보이도록 정리했습니다.

Goal브랜드 느낌과 사용 흐름이 같이 보이는 웹 화면 만들기
Process초안 비교 → 메뉴 구조 정리 → 홈 화면 완성도 조정 → 세부 화면 연결
Result홈, 카테고리, 검색, 위시리스트, 장바구니 흐름을 한 세트로 정리
ToolsHTML · CSS · JavaScript

04. Product Errors

옷 이미지 생성 결과에서 수정이 필요한 유형을 미스매치, 로고 오류, 톤앤무드로 나눠 정리했습니다.

Goal쇼핑몰에 올릴 수 있는 이미지 기준 세우기
Process대량 생성 → 이미지 확인 → 오류 유형 분류 → 다음 프롬프트 기준 정리
Result로고, 제품 형태, 색감, 브랜드 무드 기준을 따로 확인해야 함을 정리
ToolsImage Generation · Prompt Review

05. Prompt Sheet

luxury_fashion_accessory_prompt_bulk_template.xlsx는 패션 액세서리와 소품 이미지를 여러 장 만들기 위한 엑셀 프롬프트 템플릿입니다. 소재, 색감, 조명, 배경, 무드를 조합해 완성형 프롬프트를 만들고, 360도 회전 이미지용 프롬프트까지 따로 정리했습니다.

Goal상품 이미지를 한 장씩 만드는 방식에서 벗어나 반복 제작 구조 만들기
Process재료 목록 정리 → 입력 템플릿 구성 → 완성 프롬프트 생성 → 결과 이미지 검수
Result300개 프롬프트, 입력 템플릿, 360뷰 24컷, 재료표, 요약 시트로 구성
ToolsExcel · Prompt System
  • 사용법: 템플릿을 어떤 순서로 쓰는지 정리
  • 요약: 카테고리별 프롬프트 수와 구조 확인
  • 재료: 카테고리, 소재, 색, 조명, 배경, 무드 목록
  • 대량프롬프트: 바로 복사해 쓸 수 있는 완성형 문장
엑셀 템플릿 보기

06. Model Images

쇼핑몰 홈 화면에 들어갈 모델 이미지를 성공과 실패로 나눴습니다. 의상, 포즈, 배경, 브랜드 무드가 함께 맞는지 확인했습니다.

Goal홈 화면에 쓸 수 있는 모델 비주얼 고르기
Process모델 이미지 생성 → 홈 화면 적합도 확인 → 성공/실패 분류
Result과하게 튀는 결과보다 옷과 화면 분위기가 같이 맞는 이미지가 적합하다고 판단
ToolsImage Generation · Visual Curation
