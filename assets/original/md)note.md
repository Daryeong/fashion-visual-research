### `06_home screen`
- 첫 화면을 큰 이미지 중심의 에디토리얼 메인으로 구성함.

#### 미세키서울 비교 기준
- 미세키서울과 비교했을 때, 실제 사이트에는 SEARCH, SHOP IN 언어/통화 선택, WISH, CART, 로그인, 계정 생성, 비회원 주문, 소셜 로그인, 무료배송 안내, 상품 할인가/타임세일, 재고/상품문의 정보가 들어가 있음.
- 상단 카테고리도 WOMEN, MEN, COLLABORATION, COLLECTION, STYLE, ARCHIVE, ABOUT, MEMBERSHIP처럼 브랜드/쇼핑 카테고리가 세분화되어 있음.
- 지금 만든 사이트에 이미 있던 기능은 첫 화면 메인 진입, 쇼핑몰 화면 전환, 메뉴 카테고리, 검색 팝업, 상품 상세 팝업, 장바구니, 수량 변경, 삭제, 합계 계산, 찜 저장 데이터, 상품 필터/정렬, 뉴스레터, 첫 화면 돌아가기 버튼임.
- 미세키서울 기준으로 부족했던 기능은 SHOP IN, WISH 전용 창, ACCOUNT 로그인 창, 소셜 로그인 버튼, 비회원 주문/회원가입 링크, 무료배송 공지 바, 상품 할인/재고/문의 정보였음.
- 그래서 이번 파일에 해당 기능을 추가함.

#### 추가한 기능
- SHOP IN 언어/통화 선택 팝업.
- WISH 전용 찜 목록 창.
- ACCOUNT 로그인 팝업.
- 네이버/카카오/구글/Apple 소셜 로그인 데모.
- Forgot ID / Forgot Password / Create Account / Nomember Order 링크.
- 상단 무료배송/쿠폰 공지 바.
- 상품 카드에 무료배송/타임세일 배지.
- 상품 상세 팝업에 재고 수량, 상품문의, 할인 타이머 데모 표시.

## 3. 전체 기능 정리

- 가능한 기능은 팝업/새창처럼 뜨는 방식으로 변경함.
- 상품 클릭/구매 쪽은 그대로 작동함.
- 운영용 결제/재고/관리자 페이지는 아니며, 프론트엔드 데모로 정리함.
- 첫 화면 참고: 미세키서울 + MACHINE-A.
- 장바구니 Drawer: 상품 담기, 수량 +/−, 삭제, 합계 계산.
- LocalStorage 저장: 새로고침해도 장바구니/찜 목록 유지.
- Quick View 모달: 상품 사진 크게 보기 + 바로 담기.
- 찜 기능: 하트를 누르면 저장/해제.
- 검색 기능: 상품명/카테고리/색상 검색.
- 카테고리 필터: Sunglasses, Necklace, Ring, Pouch.
- 가격 정렬: 낮은 가격순 / 높은 가격순.
- 뉴스레터 이메일 입력 검증.
- Checkout Demo 버튼: 실제 결제 연동 전 단계처럼 작동.
- Shop the Collection 클릭 시 컬렉션 팝업이 열림.
- New In / Sunglasses / Accessories / Collections / Stores 메뉴 클릭 시 팝업 창이 열림.
- View All / Campaign / Installation은 페이지 이동 대신 팝업 설명창으로 열림.
- Search 버튼 클릭 시 검색 팝업창이 뜨도록 구성함.
- 기존 기능 유지: 장바구니, 수량 변경, 삭제, 합계 계산, 찜, Quick View, 검색/필터/정렬, 뉴스레터.
- 맨 위 히어로 영역 자동 전환.
- 점 버튼 클릭 시 히어로 슬라이드 변경.
- Menu 버튼 작동.
- Menu 클릭 시 새창 같은 카테고리 팝업.
- 카테고리를 많이 추가함: All, 옷, 상의, 하의, 아우터, 신발, 가방, 아이웨어, 액세서리, 뷰티, 오브젝트.
- 각 카테고리를 누르면 상품이 많이 뜨게 구성함.
- 기존 장바구니, Quick View, 찜, 검색/필터/정렬 기능 유지.
- 첫 화면: 큰 이미지 중심의 에디토리얼 메인.
- 중앙 로고 + 상단 카테고리 메뉴 구조.
- 큰 이미지 3분할 구조.
- 첫 화면을 패션 매거진 커버/목차형 구조로 구성함.
- Enter Shop, Shop Eyewear, Shop Accessories, New In을 누르면 쇼핑몰 화면으로 진입.
- 진입 후에는 기존에 만든 쇼핑몰 기능 유지.
- 상단 로고를 누르면 다시 첫 메인 화면으로 돌아가게 설정.
- LN-CC의 홈은 대형 이미지 블록, New In, 카테고리, 에디토리얼/스포트라이트 카드 구조가 중심이라 그 흐름을 참고하여 구성함.
- 첫 화면 왼쪽: 액세서리 / 소품 진입.
- 첫 화면 가운데: 메인 컬렉션 진입.
- 첫 화면 오른쪽: 리테일 오브젝트 / 디스플레이 진입.
- 베이지는 쓰지 않고 cool white, silver, charcoal, deep navy 중심으로 유지.
- SHOP IN 언어/통화 선택 팝업.
- WISH 전용 찜 목록 창.
- ACCOUNT 로그인 팝업.
- 네이버/카카오/구글/Apple 소셜 로그인 데모.
- Forgot ID / Forgot Password / Create Account / Nomember Order 링크.
- 상단 무료배송/쿠폰 공지 바.
- 상품 카드에 무료배송/타임세일 배지.
- 상품 상세 팝업에 재고 수량, 상품문의, 할인 타이머 데모 표시.
