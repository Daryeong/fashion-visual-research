# 패션 이미지 프롬프트 대량 생성 정리

## 참고 방향

고급 쇼핑몰 구조: SSENSE / Mytheresa / NET-A-PORTER / LN-CC  
비싸고 개성 강한 브랜드 무드: Rick Owens / Maison Margiela / Yohji Yamamoto / The Row  
고급인데 너무 뻔하지 않은 느낌: H.Lorenzo / Antonioli / LEMAIRE / Dries Van Noten

쇼핑몰을 참고해서 패션 액세서리와 소품 이미지를 전부 생성하고 싶다.  
이미지를 엄청나게 여러 장 뽑고 싶고, 스프레드시트로 몇백 장을 생성할 수 있도록 프롬프트 대량 생산용 엑셀 템플릿을 만들고 싶다.

---

## 엑셀 템플릿 구성

**대량프롬프트_300**: 바로 복사해서 쓸 수 있는 완성형 프롬프트 300개  
**입력템플릿**: 드롭다운으로 소재, 색감, 무드를 바꾸면 자동으로 프롬프트 생성  
**360뷰_24컷**: 상품 1개를 24컷 회전 이미지로 뽑는 전용 프롬프트  
**프롬프트재료**: 카테고리, 소재, 조명, 배경, 무드 목록  
**요약**: 카테고리별 프롬프트 개수 확인

---

## 이미지 생성 기록

### img-1st-generation 종류

처음 생성됨 / 다양하게 뽑아달라고 요청

패션 분류:  
정장, 유니폼, 수영복, 원피스, 이너, 외투, 치마, 바지 등 세부 카테고리로 구분

누끼를 따서 1개씩 10장 뽑아달라고 요청했고, 색깔도 다양하게 요청했다.

---

### img-2nd-generation 종류

깔끔하게 옷이 뽑히기 시작함.  
이후 계속 뽑아달라고 요청했다.

---

## 생성 과정에서 생긴 문제

이미지 1장씩만 뽑아달라고 했는데, 가끔씩 스프레드시트 형식으로 뽑아줄 때가 있었다.  
누끼 PNG도 제대로 되지 않았다.  
체커보드 배경이 이미지 안에 들어간 PNG가 생성될 때가 있었다.
